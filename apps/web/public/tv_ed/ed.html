Why E/W break

Side rails are perspective objects, not simple rotated rows. Use polygon_px for actual side hit/card geometry. bbox_px is an envelope only for angled targets.

Responsive rule

scale = min(viewW/1536, viewH/1024)
offsetX = (viewW - 1536*scale)/2
offsetY = (viewH - 1024*scale)/2
screenX = offsetX + designX*scale
screenY = offsetY + designY*scale
